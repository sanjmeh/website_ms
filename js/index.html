Learniat web application helps set up classes and visualise past progress.

  • Track aggregate classes and individual students progress.

    Visualize class progress at a student by student and topic by topic level after the class has ended. Drill down on students who need more attention and increase time on subtopics that have low grasp index. Create teaching content & assignments for the next class. Design home assignments that are adapted to the learning progress.

  • Manage all back-end admin work.

    Set up class sessions, seating plans, rooms, time-tables. Assign teachers to classes. Load students and pre-register them to their classes saving valuable class time in setting up technology. As web application takes care of all back end stuff important for a class, students just flip open their iPads and launch the Learniat app when they arrive at the class and it just takes them to the right class and the right seat allocated by the teacher. No setup time in class.

  • Add topics, sub topics and questions.

    At the heart of Learniat is mid lecture questioning. 4 different formats of questions can be loaded: multiple choice, matching the columns, hand written scribble, open ended text. Multiple choice type can have one or more right answers. Scribble type has variations of allowing markings on drawings that can be rapidly evaluated.

  • Monthly, weekly reports.

    See participation and grasp index reports for each class drilled down to each student and each topic and subtopic. Download or email reports to parents.

A teacher dashboard that allows unobtrusive viewing and interaction on each students class work.

Class view in Learniat shows students arranged on the screen in their seating order which helps teacher to visualise a large class easily with names and faces. In addition student class work is visible as thumbnail size icons and on touching any icon it pops up to a larger image. Teacher can interact with the student's work including annotating on it, which is fed back instantly to students.
