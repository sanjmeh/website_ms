Learniat web application helps set up classes and visualise past progress

  • Track aggregate classes and individual students' progress.

    Visualize class progress at a student by student and topic by topic level after the class has ended. Drill down on students who need more attention and increase time on subtopics that have low grasp index. Create teaching content & assignments for the next class basis the analytics received from the current class session. Make home assignments to be differentiated for each student almost automatically based on class performance.

  • Manage all back-end admin work.

    Set up class sessions, seating plans, rooms, time-tables. Assign teachers to classes. Load students and pre-register them to their classes saving valuable class time in setting up technology. As web application takes care of all back end stuff important for a class, students just open their iPads and launch the Learniat app when they arrive in the class No setup time lost in class.

  • Add topics, sub topics and questions.

    At the heart of Learniat is mid -lecture questioning. Four different formats of questions can be loaded - multiple choice, match the columns, Open ended text and Overlay scribble. Multiple choice type can have one or more right answers. Overlay scribble allows templated markings on drawings that can be rapidly evaluated through machine learning.

  • Monthly, weekly reports.

    See participation and grasp index reports for each class drilled-down to each student and each topic and subtopic. Download or email reports to parents.

A teacher's cockpit that always gives a seating order view of each student along with all ongoing class work.

Attention has been paid to each pain area of a teacher in a classroom. Visualising students in their seating order creates a powerful physical connection between their virtual actions and teacher's physical response. With dozens of events happening in a class per minute a teacher can soon be overwhelmed and the ordered seating does help a bit. Managing a large class is a breeze with Learniat.
